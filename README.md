# Quayio::Scanner

Scan quay.io for vulnerabilties in running docker containers. Implemented as sensu check.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'quayio-scanner'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install quayio-scanner
